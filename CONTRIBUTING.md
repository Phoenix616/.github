# Contributung Guidelines

In order to submit code as a pull request you must hold all the necessary copyright over the code to agree to assign your code to the license the project is under, or ‒ if you do not hold the copyright yourself ‒ provide appropriate information about the license your contribution is under so that it can be evaluated whether it is compatible witht he project's license. Please make some effort to check compatibility before submitting!

Please note that due to how generative large language models ("AI") work this previous guideline forbids you from submitting any "AI" generated code as you do not hold the copyright nor know the license the genrated code is under.
